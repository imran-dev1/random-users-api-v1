Fake Users Rest API

Free fake API for testing and prototyping for frontend designers.

-Powered by Express JS

Endpoints-

  • /user/random (GET) - for a random user.
  • /user/all (GET) - for all users.
  • /user/all/?limit=3 (GET) - you can run query parameters to limit the data.
  • /user/userId (PATCH) - for updating any user. You will have to pass all the properties through the body.
  • /user/userId (DELETE) - for deleting any user. You will have to pass the user id to delete that user.
